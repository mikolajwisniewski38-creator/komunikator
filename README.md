# Kominukator

A feature-rich web-based chat application built with React and Vite.

## Features

- **Real-time Messaging**: Instant message delivery and updates.
- **User Presence**: See who is online with active/inactive status indicators.
- **Avatar Customization**: Choose your emoji and background color.
- **Private & Global Chat**: Switch between a public global channel and private conversations.
- **Rich Media**: Support for file attachments and emojis.
- **Message Interactions**: Reply to messages and delete your own messages.

## Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Vanilla CSS with CSS Variables for theming
- **Icons**: Lucide React
- **Utils**: date-fns, clsx

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
